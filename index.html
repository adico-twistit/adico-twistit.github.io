Notes & Mostly Dev Stuff

Welcome.

I decided to add up some notes & share my latest experience with the technology stack of:

  • Titan Graph DB 1.0.0 with DynamoDB as backend
  • Gremlin-Groovy using Gremlin Server
  • Wrapped by an API Layer made of Loopback under NodeJs Which I used to execute social features over REST API (i guess it will be) focused mainly on titan graph DB.

Downloads & References

The setup is easy and consisting of a few steps: 1. install dynamo db locally 2. install titan 1.0.0 with dynamo as backend 3. for API i used loopback over nodeJs which provided easy to setup API framework
